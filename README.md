# 🎄 Navidad Nahir

Proyecto web desarrollado como una experiencia visual navideña dedicada a Nahir.

La aplicación consiste en una página interactiva con temática navideña diseñada para transmitir un mensaje especial mediante animaciones, imágenes y elementos visuales personalizados.

---

## ✨ Características

* Interfaz temática navideña.
* Diseño visual atractivo y responsivo.
* Experiencia simple y centrada en el usuario.
* Implementación completamente del lado del cliente.

---

## 🛠️ Tecnologías Utilizadas

* HTML5
* CSS3
* JavaScript

## 🎁 Objetivo

Crear una experiencia web sencilla y personalizada con temática navideña, priorizando la estética, la interacción y la experiencia de usuario.
